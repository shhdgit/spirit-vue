# 桌游精灵

ptt桌游精灵（Vue 2.x版）[预览](http://rollaround.cn/spirit)

## Dependencies

node、npm

## Development

* **clone至本地:**
```
git clone git@github.com:shhdgit/spirit-vue.git
```
* **安装依赖:**
```
npm install
```
* **运行开发环境:**
```
npm run dev
```
浏览器输入：localhost:8080 访问开发服务器
* **构建、打包:**
```
npm run build
```
